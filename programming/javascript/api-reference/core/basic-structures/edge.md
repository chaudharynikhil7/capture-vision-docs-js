# Edge

The `Edge` interface represents an edge defined by two [Corners](./corner.md).

```typescript
interface Edge {
    startCorner: Corner;
    endCorner: Corner;
}
```

## startCorner

The starting corner of the edge.

## endCorner

The ending corner of the edge.
