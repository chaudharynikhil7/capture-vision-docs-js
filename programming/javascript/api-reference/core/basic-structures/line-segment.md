# LineSegment

The `LineSegment` interface represents a line segment defined by two [Points](./point.md).

```typescript
interface LineSegment {
    startPoint: Point;
    endPoint: Point;
} 
```

## startPoint

The starting point of the line segment.

## endPoint

The ending point of the line segment.
